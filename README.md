# hexlet-vibecoding

Учусь вайбкодингу, тут будут мои эксперименты.

## Калькулятор калорий

Простое веб-приложение для расчёта суточной нормы калорий и макронутриентов.

### Возможности

- Расчёт базового обмена веществ (BMR) по формуле **Миффлина — Сан Жеора**
- Учёт уровня физической активности (TDEE)
- Корректировка нормы под цель: похудение, поддержание или набор массы
- Рекомендации по белкам, жирам и углеводам

### Запуск

Откройте файл `index.html` в браузере — приложение работает полностью на стороне клиента, без сборки и зависимостей.

### Файлы

- `index.html` — разметка и форма ввода
- `style.css` — стили
- `script.js` — логика расчёта
